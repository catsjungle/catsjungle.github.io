Canvas的坐标系

canvas 2d 坐标系

var can = document.getElementById('can');
var context = can.getContext('2d');
context.getContext('2d'); // 获取2d上下文
绘制矩形
context.strokeRect(x, y, width, height); // 绘制一个矩形边框
context.fillRect(x, y, width, height); // 绘制一个填充的矩形
context.clearRect(x, y, width, height); //清除一个矩形区域

绘制图片
context.drawImage(oImg, x, y);

绘制路径
context.beginPath();
context.closePath();
stroke() 通过线条绘制图形轮廓
fill() 填充路径生成实心图形
moveTo(x, y)
lineTo(x, y)
arc(x, y, radius, startAngle, endAngle, anticlockwise)

x,y为圆心坐标，radius为半径，startAngle和endAngle定义了开始和结束的弧度，anticlockwise为true,是逆时针，为false
